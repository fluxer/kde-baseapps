Pâté

A Kate plugin which provides support for Python plugins. For installation
instructions see INSTALL.txt.

Pâté was originally created by Paul Giannaros <paul@giannaros.org>; thanks
Paul! The original sources are available at:

- For KDE3 Kate, http://paul.giannaros.org/pate/

- For KDE4 Kate, https://github.com/pag/pate/

The code here is derived from the KDE4 Kate support.

Enjoy Pâté!
